Image Processing and Computer Vision Toolbox (IPCV) for Scilab 6

Version 5.0.0 (07-Jul-26)
OpenCV 5 and C++ Migration
1. Target OpenCV/OpenCV contrib 5.0.0 for thirdparty builds
2. Update gateway linking and C++ compile flags for OpenCV 5
3. Rename source DLL target from libipcv_decode to libipcv_core
4. Split source C++ into category files under src/cpp
5. Rewrite Scilab/native image exchange for the current Scilab API
6. Add OpenCV Zoo model downloader GUI with on-demand ONNX downloads
7. Exclude bundled ONNX models from distribution; demos/tests download trusted models on demand
8. Add OpenCV 5 DNN helpers and examples, including ONNX and CLIP-related examples
9. Archive old IPCV 4.5 unit tests in tests/unit_tests/4.5 and add current migration tests in tests/unit_tests
